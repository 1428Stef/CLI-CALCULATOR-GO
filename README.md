# CLI-CALCULATOR

A simple calculator written in **GO** that performs basic arithmetic operations.

## Features

- `+` Addition of numbers
- `-` Subtraction of numbers  
- `*` Multiplication of numbers
- `/` Division of numbers (with floating-point support)

## Installation

1. Make sure Go (version 1.XX or later) is installed.
2. Clone the repository.
3. Install dependencies:

```bash
go get github.com/1428Stef/CLI-CALCULATOR-GO
```

## Run the program:

```bash
go run main.go
```
